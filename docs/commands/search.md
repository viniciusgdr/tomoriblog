# Comandos de Busca

Comandos para buscar informacoes e identificar conteudo.

## /pinterest

Busca imagens no Pinterest.

| Propriedade | Valor |
|-------------|-------|
| **Aliases** | `pin` |
| **Permissao** | Todos |

**Uso:**
```
/pinterest wallpaper anime
/pin natureza 4k
```

Retorna uma imagem aleatoria dos resultados da busca.

---

## /ytsearch

Busca videos no YouTube.

| Propriedade | Valor |
|-------------|-------|
| **Aliases** | `yts` |
| **Permissao** | Todos |

**Uso:**
```
/ytsearch tutorial programacao
```

Retorna uma lista de videos com titulo e link.

---

## /shazam

Identifica musicas a partir de audio.

| Propriedade | Valor |
|-------------|-------|
| **Aliases** | `musica` |
| **Permissao** | Todos |

**Uso:**
```
/shazam (respondendo um audio ou video)
```

Aceita tanto audio quanto video (video e convertido para audio automaticamente).
