# Modulos de Comandos

O sistema de modulos permite ativar ou desativar comandos individualmente por grupo, e restringir comandos apenas para administradores.

## Como acessar

1. Acesse o **painel do grupo**
2. Va em **Modulos**

## Funcionalidades

### Ativar/Desativar comandos

Cada comando tem um **toggle** para ligar ou desligar no grupo. Quando desativado, o comando nao responde no grupo.

### Restringir a admins

Cada comando tem um toggle de **"Privado para admins"** que restringe o uso apenas a administradores do grupo. Este toggle so funciona quando o comando esta ativado.

### Mensagem de comando indisponivel

No topo da pagina, ha um toggle global para **"Mostrar mensagem quando comando indisponivel"**. Quando ativado, o bot notifica o membro de que o comando esta desativado no grupo.

## Busca e filtro

- Use o campo de **busca** para filtrar comandos por nome ou descricao
- Use o **filtro de categoria** para ver apenas comandos de uma categoria especifica

## Informacoes exibidas

Cada card de comando mostra:

| Info | Descricao |
|------|-----------|
| **Nome** | Nome do comando |
| **Descricao** | O que o comando faz |
| **Categoria** | Categoria do comando (IA, Diversao, etc) |
| **Apenas Admin** | Badge se o comando e exclusivo para admins |
| **Bot Admin** | Badge se o bot precisa ser admin |

## Exemplos de uso

- Desativar comandos de diversao em grupo profissional
- Restringir downloads apenas para admins
- Desativar jogos em dias de semana e ativar nos finais de semana
