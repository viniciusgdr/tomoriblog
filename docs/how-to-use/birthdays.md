# Aniversarios

O TomoriBOT possui um sistema completo de gerenciamento de aniversarios para grupos.

## Funcionamento

- Membros podem cadastrar seus aniversarios
- O bot notifica o grupo quando e aniversario de alguem
- Lista de proximos aniversarios disponivel via comando e painel

## Comando

```
/aniversario
```

Alias: `aniversario`

Exibe os proximos aniversarios do grupo.

## Gerenciamento pelo painel

O painel web oferece um gerenciamento completo de aniversarios:

1. Acesse o [painel de controle](https://tomoribot.com.br/dashboard)
2. Selecione o grupo
3. Na aba de membros ou configuracoes, gerencie aniversarios

### Funcionalidades do painel

- Adicionar aniversario de um membro
- Editar data de aniversario
- Remover aniversario
- Ver lista completa de aniversarios do grupo
- Configurar notificacoes automaticas

## Notificacoes

Quando configurado, o bot envia uma mensagem automatica no grupo no dia do aniversario do membro, parabenizando-o.
