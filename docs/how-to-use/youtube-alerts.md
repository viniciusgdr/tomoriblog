# Alertas de YouTube

Configure alertas automaticos para que o bot notifique o grupo quando um canal do YouTube publicar um novo video.

## Como configurar

1. Acesse o **painel do grupo**
2. Va em **Alertas de Canal do YouTube**
3. Insira o **ID do canal** do YouTube
4. Clique em **Adicionar**

## Como encontrar o ID do canal

1. Acesse o canal do YouTube
2. O ID do canal esta na URL: `youtube.com/channel/UCxxxxxxx`
3. Copie o codigo que comeca com `UC`

## Gerenciamento

Os alertas configurados aparecem em uma tabela com:

| Coluna | Descricao |
|--------|-----------|
| **ID do canal** | Link clicavel para o canal |
| **Data de criacao** | Quando o alerta foi criado |
| **Acoes** | Editar ou Excluir |

## Exemplos de uso

- Alertar o grupo sobre novos videos de aulas
- Acompanhar canais de noticias
- Receber notificacoes de conteudo religioso
