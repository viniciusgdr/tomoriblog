# Loja do Grupo

A loja do grupo permite que administradores criem itens que membros podem comprar usando a moeda virtual (Cash). Itens podem conceder emblemas (badges) especiais.

## Como acessar

1. Acesse o **painel do grupo**
2. Va em **Loja**

## Abas disponiveis

### Itens

Crie e gerencie itens da loja.

| Campo | Descricao |
|-------|-----------|
| **Nome** | Nome do item |
| **Descricao** | O que o item faz ou representa |
| **Preco** | Valor em Cash |
| **Emblema vinculado** | Badge concedido ao comprar (opcional) |

### Emblemas

Crie emblemas (badges) que aparecem no perfil dos membros.

| Campo | Descricao |
|-------|-----------|
| **Emoji/Texto** | Emoji ou texto do badge |

Cada emblema mostra quantos membros o possuem.

### Historico de Compras

Registro de todas as compras realizadas com item, membro e data.

## Como funciona

1. Admin cria itens no painel com nome, descricao e preco
2. Membros usam o comando `/loja` no grupo para ver os itens
3. Ao comprar, o Cash e descontado do saldo do membro
4. Se o item tem um emblema vinculado, ele e concedido automaticamente
5. Se o saldo for insuficiente, a compra e recusada

## Dicas

- Use emblemas para criar um sistema de status no grupo
- Crie itens com precos progressivos para incentivar atividade
- Acompanhe as compras pelo historico para entender o engajamento
