# Bots

A Tomori aceita você alocar seu número e utilizar nossa infraestrutura para hospedar no seu número. Para isso, você precisa de um bot.

![Example banner](/img/bots.png)

## Vinculando um bot

1. Crie um bot e informe o número de telefone que deseja vincular.
2. Inicie o bot e vincule com o código de pareamento que será mostrado no painel.

### Vinculando no WhatsApp

1. Abra o WhatsApp
2. Configurações
3. Dispositivos conectados
4. Vincular um dispositivo
5. Clique na parte inferior da tela "Vincular um dispositivo"
6. Digite o código de pareamento
