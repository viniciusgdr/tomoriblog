# Mensagens Agendadas

Programe mensagens automaticas que o bot envia no grupo em dias e horarios configurados, com texto, midia e mencao geral.

:::info
Este recurso requer **plano Dedicado**.
:::

## Como criar

1. Acesse o **painel do grupo**
2. Va em **Mensagens Agendadas**
3. Clique em **Criar**
4. Preencha o formulario

## Campos do formulario

| Campo | Descricao |
|-------|-----------|
| **Nome da campanha** | Identificacao da mensagem |
| **Texto** | Conteudo da mensagem |
| **Tipo de midia** | Somente Texto ou Imagem |
| **Upload de midia** | Imagem para enviar junto (se tipo = Imagem) |
| **Dias da semana** | Selecao multipla (seg a dom) |
| **Hora e Minuto** | Horario do envio |
| **Mencionar todos** | Marca @todos na mensagem |

## Exemplos de uso

- Lembrete diario de regras do grupo as 08:00
- Aviso semanal de reuniao toda segunda as 09:00
- Mensagem de bom dia com imagem de segunda a sexta

## Painel lateral

Na pagina de mensagens agendadas, o painel lateral exibe os **proximos 5 disparos** programados com contagem regressiva, para voce acompanhar quando sera o proximo envio.

## Gerenciamento

As mensagens aparecem em uma tabela interativa com filtros e ordenacao. Cada mensagem pode ser editada ou excluida.
