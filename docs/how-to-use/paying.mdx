# Gerenciando os Grupos

O painel de controle é a interface web onde você pode gerenciar seus grupos, usuários e mensagens. Para acessar o painel de controle, acesse [aqui](https://tomoribot.gdr.dev.br/dashboard).

## Grupos

Para gerenciar ou adicionar um novo grupo, clique na aba "Grupos" no menu lateral. Você verá uma lista de todos os grupos que você tem acesso. Para adicionar um novo grupo, clique no botão "Adicionar Grupo" no canto superior direito.

![Example banner](/img/groups.png)

### Para adicionar um novo grupo

1. Clique no botão "Adicionar Grupo" no canto superior direito.

![Example banner](/img/add_group.png)

2. Preencha o formulário com o link do grupo.

### Para gerenciar um grupo

Na tela de "Membros", você pode ver todos os membros do grupo e suas informações. Você pode adicionar ou remover membros, alterar o nível de acesso de um membro e ver as mensagens enviadas pelo grupo.

Você também pode banir os membros que não enviaram mensagens por um determinado período de tempo.

![Example banner](/img/group-member.png)

### Ativações

As funcionalidades do bot são ativadas por meio de ativações. Para ativar uma funcionalidade, clique no botão "Ativar" ao lado da funcionalidade desejada.

![Example banner](/img/activations.png)

### Comandos

Na aba de comandos, você pode definir comandos personalizados, adicionar usuários em uma blacklist, configurar exceções do antilink, definir regras e até mesmo automatizar a abertura e o fechamento do grupo.
![Example banner](/img/commands.png)

### Configurações

Na aba de configurações, você pode remover o bot do grupo, limpar os dados e zerar o contador.

![Example banner](/img/config.png)
