# Jogos de Palavras

Jogos que testam seu vocabulario e conhecimento.

## Anagrama

Desembaralhe as letras para formar a palavra correta.

```
/anagrama
```

### Regras

- Voce recebe letras embaralhadas
- **3 tentativas** para acertar
- **2 minutos** de tempo limite
- Ganhe Cash ao acertar

### Para responder

```
/anagrama <sua_resposta>
```

---

## Forca

O classico jogo da forca.

```
/forca
```

### Regras

- **8 tentativas** antes do boneco ser completo
- Pode chutar letra por letra ou a palavra inteira
- Boneco visual em emojis (completo: cabeça, tronco, bracos e pernas)

### Recompensas

| Resultado | Cash |
|-----------|------|
| Acertou | +100 |
| Errou | -50 |

### Para jogar

```
/forca a
/forca banana
```

## Perguntados

Quiz de multipla escolha.

```
/perguntados
```

### Como funciona

1. O bot envia uma pergunta com 4 opcoes (A, B, C, D)
2. Responda com a letra da alternativa
3. Ganhe pontos ao acertar

As perguntas cobrem diversos temas de conhecimento geral.

## Quem Santo

Adivinhe o santo pelas pistas (veja [Comandos Catolicos](../catholic/overview)).

```
/quemsanto
```

Jogo por grupo - apenas um jogo ativo por vez. Dicas aleatorias a cada tentativa errada.

## Espelho

Inverta a frase corretamente e ganhe Cash.

```
/espelho
```

### Como jogar

1. Use `/espelho` para iniciar — o bot mostra uma frase invertida
2. Responda com `/espelho <frase original>`
3. Se acertar, ganha **+100 Cash**
4. Se errar, perde **-80 Cash**

### Regras

- Tempo limite: **5 minutos** para responder
- Apenas um jogo ativo por vez
