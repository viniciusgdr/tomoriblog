# Jogos de Tabuleiro

Jogos classicos de tabuleiro adaptados para o WhatsApp.

## Batalha Naval

Afunde todos os navios do adversario em um grid 5x5.

```
/batalhanaval
```

### Como jogar

1. Inicie o jogo com `/batalhanaval`
2. Posicione seus navios informando coordenadas (ex: A1, B3)
3. Reveze turnos atacando posicoes

### Grid

Posicoes de A1 ate E5:

| | 1 | 2 | 3 | 4 | 5 |
|---|---|---|---|---|---|
| **A** | A1 | A2 | A3 | A4 | A5 |
| **B** | B1 | B2 | B3 | B4 | B5 |
| **C** | C1 | C2 | C3 | C4 | C5 |
| **D** | D1 | D2 | D3 | D4 | D5 |
| **E** | E1 | E2 | E3 | E4 | E5 |

### Legenda

| Emoji | Significado |
|-------|-------------|
| ⬜ | Celula vazia |
| 💣 | Bomba (sua) |
| 🌊 | Tiro na agua (miss) |
| 💥 | Acertou navio (hit) |

Funciona no grupo e no privado.

## Campo Minado

Revele todas as celulas sem acertar as bombas.

```
/campominado
```

### Como jogar

1. Inicie com `/campominado`
2. Escolha posicoes para revelar (ex: A1, C3)
3. Evite as bombas

### Legenda

| Emoji | Significado |
|-------|-------------|
| ⬜ | Celula oculta |
| ✅ | Celula revelada (segura) |
| 💥 | Bomba (game over) |
| 🚩 | Bandeira marcada |

Funciona no grupo e no privado.

## Jogo da Velha (/ttt)

Jogue contra outro membro do grupo.

```
/ttt
```

### Como funciona

- Tabuleiro visual renderizado como imagem (1080x1080)
- Um jogador e X, outro e O
- Revezamento de jogadas
- Funciona no grupo e no privado

O tabuleiro e enviado como imagem/sticker a cada jogada.
