# Guia para Iniciantes

Nunca usou um bot no WhatsApp? Sem problemas. Este guia explica tudo do zero.

## O que e o TomoriBOT?

O TomoriBOT e um **robô automatico** que funciona dentro dos seus grupos do WhatsApp. Ele responde a comandos que voce digita no chat, executando funcoes como:

- Criar figurinhas a partir de imagens
- Baixar musicas e videos
- Jogar jogos com os amigos
- Moderar o grupo automaticamente
- Conversar com inteligencia artificial
- E muito mais (120+ funcionalidades)

## Como funciona?

1. Voce adiciona o bot ao seu grupo do WhatsApp
2. Digita um **comando** no chat (ex: `/sticker`)
3. O bot responde automaticamente

Um **comando** e uma mensagem que comeca com `/` (barra) seguido de uma palavra. Por exemplo:

| Voce digita | O bot faz |
|-------------|-----------|
| `/sticker` (com uma imagem) | Transforma a imagem em figurinha |
| `/play despacito` | Envia a musica do YouTube |
| `/gpt o que e um buraco negro?` | Responde usando inteligencia artificial |
| `/clima Sao Paulo` | Mostra o clima atual |

## Passo a passo: do zero ao primeiro comando

### 1. Crie sua conta

Acesse [tomoribot.com.br/account/register](https://tomoribot.com.br/account/register) e crie uma conta com seu email.

### 2. Escolha um plano

O TomoriBOT e um servico pago. Veja os planos em [tomoribot.com.br/#pricing](https://tomoribot.com.br/#pricing) e escolha o que melhor atende ao seu grupo.

Saiba mais em [Planos e Limites](./plans).

### 3. Adicione o bot ao grupo

Depois de assinar, siga o guia completo em [Adicionando o Bot ao Grupo](./adding-bot).

**Resumo rapido:**
- Adicione o numero do bot como participante do grupo
- Ative o grupo no [painel de controle](https://tomoribot.com.br/dashboard)
- Promova o bot a admin do grupo (para comandos de moderacao)

### 4. Envie seu primeiro comando

No grupo, digite:

```
/menu
```

O bot vai responder com a lista de todas as categorias de comandos disponiveis.

### 5. Experimente os comandos mais populares

| Comando | O que faz | Como usar |
|---------|-----------|-----------|
| `/sticker` | Cria figurinha | Envie uma imagem e escreva `/s` na legenda |
| `/play` | Baixa musica | `/play nome da musica` |
| `/gpt` | Conversa com IA | `/gpt sua pergunta aqui` |
| `/captura` | Captura Pokemon | `/captura` (quando aparecer um Pokemon) |
| `/perfil` | Veja seu perfil | `/perfil` |
| `/ranking` | Ranking do grupo | `/ranking` |
| `/casal` | Forma casal aleatorio | `/casal` |

## Dicas importantes para iniciantes

### Prefixo

O prefixo padrao e `/` (barra). Todos os comandos comecam com ela. Se nao funcionar, o admin pode ter mudado para outro prefixo (como `!` ou `.`).

### Aliases

Muitos comandos tem **nomes curtos**. Voce nao precisa digitar o nome completo:

- `/sticker` = `/s` = `/f` = `/fig`
- `/play` = `/p`
- `/instagram` = `/ig`

### Respondendo mensagens

Alguns comandos funcionam **respondendo** uma mensagem. Por exemplo:
- Responda uma imagem com `/s` para transforma-la em figurinha
- Responda um audio com `/transcrever` para converter em texto
- Responda um video com `/tomp3` para extrair o audio

### Comandos no privado

Não é suportado.

### O bot nao respondeu?

Verifique:
1. O grupo esta **ativado** no painel?
2. O comando esta **ativado** nos [Modulos](../how-to-use/modules)?
3. O prefixo esta correto?
4. O bot esta no grupo como participante?
