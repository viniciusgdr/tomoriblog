# Adicionando o Bot ao Grupo

Existem duas formas de usar o TomoriBOT: atraves de um bot compartilhado ou vinculando seu proprio numero.

:::info
A vinculacao de bots (usar seu proprio numero como bot) e exclusiva para **planos Dedicados**. Nos demais planos, voce utiliza um bot compartilhado ja disponibilizado pelo TomoriBOT.
:::

## Usando bot compartilhado

1. Acesse o [painel de controle](https://tomoribot.com.br/dashboard)
2. Vá na [tela de Grupos](https://tomoribot.com.br/dashboard/groups)
3. Insira o link do grupo e selecione o BOT.
4. O Bot irá entrar no grupo automaticamente. Envie `/menu` no grupo para ver todos os comandos

OBS: Caso der erro nessa parte, chame o suporte.

## Vinculando seu proprio bot (Plano Dedicado)

Se voce possui um plano Dedicado, pode vincular seu proprio numero:

### 1. Vincule seu bot

Siga o guia de [Vinculacao de Bots](../how-to-use/bots) para parear seu numero com o TomoriBOT.

### 2. Adicione ao grupo

2. Vá na [tela de Grupos](https://tomoribot.com.br/dashboard/groups)
3. Insira o link do grupo e selecione o BOT.

### 3. Primeiro comando

Envie `/menu` no grupo para ver todos os comandos disponiveis.

## Promova a admin

Para que comandos de moderacao funcionem (banir, fechar grupo, etc), o bot precisa ser **admin do grupo**.

> Sem admin, comandos como `/ban`, `/grp`, `/anunciar` e `/antighost` nao funcionarao.

## Dicas importantes


- O prefixo padrao e `/` (barra), mas pode ser alterado no painel
- O bot responde apenas em grupos onde esta ativado (exceto comandos que funcionam no privado)
- Comandos marcados como "Admin" so podem ser usados por admins do grupo
- Comandos marcados como "Bot Admin" exigem que o bot seja admin
