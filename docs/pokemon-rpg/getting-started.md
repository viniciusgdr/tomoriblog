# Comecando no RPG Pokemon

O TomoriBOT possui um sistema completo de RPG Pokemon integrado ao WhatsApp. Capture, treine, batalhe e colecione Pokemon diretamente nos seus grupos.

## Seu primeiro Pokemon

Use `/pkminicial` para escolher seu Pokemon inicial:

```
/pkminicial
```

Isso mostrara as 3 opcoes disponiveis:

| Pokemon | Tipo |
|---------|------|
| Charmander | Fogo |
| Bulbasaur | Planta |
| Squirtle | Agua |

Para escolher, envie o nome:

```
/pkminicial charmander
```

:::caution Atencao
Voce so pode escolher **uma vez**. Escolha com sabedoria!
:::

## Proximo passo: capturar Pokemon

Apos escolher seu inicial, voce ja pode comecar a capturar Pokemon selvagens com `/captura`.

## Visao geral do sistema

```
Escolher inicial (/pkminicial)
       |
       v
Capturar Pokemon (/captura)
       |
       v
Treinar e subir de nivel (batalhas)
       |
       v
Desafiar ginasios (/ginasio)
       |
       v
Viajar para novas regioes (/novaregiao)
       |
       v
Completar a Pokedex (/pokedex)
```

## Comandos essenciais

| Comando | Funcao |
|---------|--------|
| `/pkminicial` | Escolher Pokemon inicial |
| `/captura` | Capturar Pokemon selvagem |
| `/pokedex` | Ver progresso de captura |
| `/inventario` | Ver e gerenciar seus Pokemon |
| `/batalha @user` | Batalhar contra outro jogador |
| `/batalha-npc` | Batalhar contra NPC da regiao |
| `/ginasio` | Desafiar lider de ginasio |
| `/novaregiao` | Viajar para outra regiao |
| `/loja` | Comprar itens |
| `/banco` | Ver mercado de trocas |
| `/wondertrade` | Trocar Pokemon aleatoriamente |
| `/tempo` | Ver cooldowns |
