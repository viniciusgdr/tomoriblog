# Ginasios e Regioes

## Ginasios

Use `/ginasio` (aliases: `gym`, `ginasios`) para desafiar lideres de ginasio.

```
/ginasio
```

### Lideres classicos

Os lideres seguem a estrutura classica de Pokemon:

| Lider | Tipo | Insignia |
|-------|------|----------|
| Brock | Pedra | Boulder |
| Misty | Agua | Cascade |
| Lt. Surge | Eletrico | Thunder |
| Erika | Planta | Rainbow |
| Koga | Veneno | Soul |
| Sabrina | Psiquico | Marsh |
| Blaine | Fogo | Volcano |
| Giovanni | Terra | Earth |

### Regras

- **1 tentativa por dia**
- Cada lider tem um time de Pokemon com stats definidos
- Vencer concede insignia + recompensas em Cash e XP
- Insignias sao exibidas no seu `/inventario`

## Regioes

Use `/novaregiao` (aliases: `viajar`, `travel`, `regiao`) para viajar entre regioes.

### Listar regioes

```
/novaregiao
```

Sem argumento, lista todas as regioes disponiveis com seus requisitos de nivel.

### Viajar

```
/novaregiao kanto
/novaregiao johto
```

### Regras

- **Cooldown:** 60 minutos entre viagens
- Cada regiao tem um **nivel minimo** para acesso
- Regioes diferentes possuem Pokemon diferentes
- Seu progresso na Pokedex mostra captura por regiao

### Por que viajar?

- Encontrar Pokemon exclusivos de cada regiao
- Completar a Pokedex
- Enfrentar NPCs diferentes
- Acessar ginasios de cada regiao
