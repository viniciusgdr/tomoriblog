# Programa de Indicacao

Ganhe comissoes indicando novos clientes para o TomoriBOT. Receba **12% vitalicia** sobre as assinaturas dos seus indicados.

## Como funciona

1. Compartilhe seu **link de indicacao** com outras pessoas
2. Quando alguem assinar usando seu link, voce ganha comissao
3. A comissao e de **12%** do valor da assinatura, **vitalicia**
4. Quando acumular saldo suficiente, solicite um saque via PIX

## Como acessar

1. Acesse [tomoribot.com.br/dashboard](https://tomoribot.com.br/dashboard)
2. Va em **Revenda**

## Painel de estatisticas

| Metrica | Descricao |
|---------|-----------|
| **Comissao total** | Valor total acumulado de comissoes |
| **Saldo disponivel** | Valor disponivel para saque |
| **Taxa de comissao** | 12% vitalicia |

## Link de indicacao

Na pagina, voce encontra:
- **Codigo de indicacao** — para compartilhar manualmente
- **Link de indicacao** — link direto para compartilhar

Ambos possuem botao de copiar.

## Solicitar saque

1. Clique em **Solicitar Saque**
2. Informe o valor desejado
3. Informe sua **chave PIX**
4. Confirme

### Regras de saque

| Regra | Valor |
|-------|-------|
| Valor minimo | R$ 15,00 |
| Limite maximo | Saldo disponivel |
| Chave PIX | Obrigatoria |
