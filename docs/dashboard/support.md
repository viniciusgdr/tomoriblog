## Perguntas Frequentes

### Como configurar o bot?

Apos assinar um plano, acesse o [painel de controle](https://tomoribot.com.br/dashboard) e siga o guia de [Adicionando o Bot ao Grupo](../getting-started/adding-bot).

### Qual o uptime do servico?

O TomoriBOT opera com alta disponibilidade. Em caso de manutencoes programadas, avisamos com antecedencia nos canais oficiais.

### Posso mudar de plano?

Sim. Entre em contato com o suporte para realizar a mudanca de plano.

### Meu numero foi banido, e agora?

Se o numero do bot foi banido pelo WhatsApp, entre em contato com o suporte para orientacoes sobre como proceder.

### O bot nao responde no meu grupo

Verifique se:
1. O grupo esta ativado no painel
2. O bot esta no grupo como participante
3. O comando utilizado esta ativado nos [Modulos](../how-to-use/modules)
4. O prefixo esta correto (padrao: `/`)
