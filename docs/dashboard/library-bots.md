# Biblioteca de Bots IA

Crie, edite e explore bots de IA personalizados. Cada bot e uma "personalidade" com prompt customizado que define como a IA se comporta.

## Como acessar

1. Acesse [tomoribot.com.br/dashboard](https://tomoribot.com.br/dashboard)
2. Va em **Biblioteca de Bots IA**

## Meus Bots

Secao com seus bots privados. Cada card mostra:
- Nome e descricao
- Badges indicando funcoes ativas (Diversao, Anti-link, etc.)
- Botoes de **Editar** e **Ver Detalhes**

## Criar novo bot

Clique em **Criar Novo Bot** e preencha:

| Campo | Descricao |
|-------|-----------|
| **Nome** | Nome do bot |
| **Descricao** | O que o bot faz |
| **Publico/Privado** | Se visivel na comunidade ou apenas para voce |
| **Prompt IA** | Instrucao/personalidade da IA |

### Toggles de funcionalidades

Cada bot pode ter ativacoes individuais:

| Toggle | Descricao |
|--------|-----------|
| Modo de Diversao | Bot responde com humor e descontracao |
| Anti-ViewOnce | Salva mensagens de visualizacao unica |
| Anti-Delete | Recupera mensagens apagadas |
| Anti-Delete V2 | Versao avancada da recuperacao |
| Anti-Link | Bane quem envia links |
| Anti-Link WhatsApp | Bane quem envia links do WhatsApp |
| Auto Sticker | Converte imagens automaticamente em figurinhas |
| Anti Videos/Imagens | Bloqueia envio de midia |
| Auto Texto | Transcreve audios para texto automaticamente |
| Anti Pagamento | Bloqueia comprovantes de pagamento (anti-golpe) |
| Anti NSFW | Detecta e bane imagens pornograficas |
| Anti Estrangeiros | Bane numeros nao-brasileiros |
| Boas-vindas | Mensagem automatica para novos membros |
| Datas Comemorativas | Mensagens em datas especiais |
| Modo de Noticias | Envia noticias periodicamente |

## Bots da Comunidade

Explore bots publicos criados por outros usuarios. Voce pode:
- Buscar por nome ou descricao
- **Curtir** bots com o botao de coracao
- **Ver Detalhes** para mais informacoes
