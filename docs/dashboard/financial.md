# Financeiro

A pagina Financeiro permite acompanhar o status da sua assinatura, ver faturas pendentes e realizar pagamentos.

## Como acessar

1. Acesse [tomoribot.com.br/dashboard](https://tomoribot.com.br/dashboard)
2. Va em **Financeiro**

## Com plano ativo

Se voce possui um plano ativo, a pagina exibe:

| Informacao | Descricao |
|------------|-----------|
| **Status de pagamento** | Em dia, Fatura disponivel ou Pagamentos em atraso |
| **Metodo de pagamento** | PIX (unico metodo disponivel) |
| **Ultimo pagamento** | Data do ultimo pagamento realizado |

### Pagar fatura

Se houver fatura pendente no mes:

1. Selecione **PIX** como metodo de pagamento
2. Clique em **Pagar**
3. Voce sera redirecionado para a pagina de checkout com o QR Code PIX

## Sem plano

Se voce ainda nao possui um plano, a pagina exibe opcoes de planos disponiveis para assinatura.
